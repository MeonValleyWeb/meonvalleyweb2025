# Security Policy

Thank you for helping keep Meon Valley Web secure. We take security issues
seriously and will respond as quickly as possible.

## Reporting a Vulnerability

Please report security issues by email:

- <hello@meonvalleyweb.com>

Include as much detail as possible:

- A clear description of the issue and potential impact
- Steps to reproduce
- Any proof-of-concept code or screenshots (if applicable)

We will acknowledge receipt within 3 business days and provide a status update
as we investigate.

## Supported Versions

Security updates are applied to the default branch (`main`). Deployments are
made from `main` to production.
